💳 Credit Card Fraud Detection System

**Machine Learning** project that detects fraudulent credit card transactions using a scalable cloud-based data pipeline. The project leverages **Microsoft Azure**, **Docker**, and **Power BI** to build, deploy, and visualize fraud detection workflows for efficient and data-driven decision making.

---

## 🚀 Features

- Detects fraudulent credit card transactions using Machine Learning.
- Performs data preprocessing and feature engineering.
- Builds a scalable cloud-based data processing pipeline.
- Containerized deployment using Docker.
- Interactive Power BI dashboard for fraud analytics.
- End-to-end workflow from data ingestion to visualization.

---

## 🛠️ Tech Stack

- **Programming:** Python
- **Libraries:** NumPy, Pandas, Scikit-learn
- **Cloud:** Microsoft Azure, Azure Data Factory, Azure Databricks, Azure SQL Database
- **Deployment:** Docker
- **Visualization:** Power BI
- **Version Control:** Git, GitHub
